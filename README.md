# TradingAgentClassicRL

Репозиторий для курсовой: сравнение классических алгоритмов обучения с подкреплением в задаче алгоритмической торговли.

Коротко:
- Цель: честно сравнить табличные RL-алгоритмы на одной и той же торговой среде.
- Среда: дискретные признаки (RSI, MACD, тренд, позиция, PnL-состояние) + комиссии, проскальзывание и риск‑ограничения.
- Данные: исторические свечи `BTC/USDT` (таймфреймы `1h`, `15m`, `1m`).

## Среда и постановка
**Действия (3):**
- `0` — Hold
- `1` — Buy (открыть позицию)
- `2` — Sell (закрыть позицию)

**Состояние (дискретное):**
- RSI (3 уровня)
- MACD diff (3 уровня)
- MA‑тренд (3 уровня)
- Флаг позиции (0/1)
- PnL‑состояние (в минусе / около нуля / в плюсе)

Итого: 3×3×3×2×3 = **162 состояния**.

**Особенности среды:**
- Комиссия и проскальзывание.
- Ограничение по времени удержания позиции.
- Ограничение по максимальной просадке.
- Награда учитывает результат сделки и итоговый доход, с усилением штрафов за убытки.

## Алгоритмы
**Классические RL:**
- Q‑Learning
- SARSA
- SARSA($\lambda$)
- Monte‑Carlo Control

**Базовые стратегии:**
- Random
- Buy&Hold
- Moving Average

## Метрики
- Средняя награда и дисперсия
- Средняя стоимость портфеля
- Количество сделок
- Win Rate
- Profit Factor
- Total / Avg PnL


## Как воспроизвести
Минимальный пример (см. `notebooks/test_train.ipynb`):

```python
import pandas as pd
from training import TrainingManager, TrainingConfig
from agents.classical.qlearning_agent import QLearningAgent

# данные
df = pd.read_csv("data/data_1h_2021.csv")

# агент и конфиг
agent = QLearningAgent()
config = TrainingConfig(agent_name="QLearning", agent_type="QLearning")

# обучение
manager = TrainingManager()
manager.train_agent(agent, df, config, experiment_name="qlearning_1h_2021")
```
